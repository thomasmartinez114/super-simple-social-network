# Super Simple Social Network

## Backend

Node.js (node server.js)

- Dependencies:
  - body-parser
  - cors
  - dotenv
  - express
  - mysql2

## Frontend

Vite/React (npm run start_full_app [concurrently command])

- Dependencies:
  - axios
  - Moment (date and time format)
  - React-router-dom
  - react-icons
  - concurrently
    - updated the package.json
